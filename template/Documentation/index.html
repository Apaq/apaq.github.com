Installing The Template

Setting the base color scheme

  1. Locate css/colors folder, you have all the colors in there.
  2. Choose your color.
  3. Open the html files.
  4. Go to Line 40 and replace the color name with your chosen color.
<link rel="stylesheet" type="text/css" href="css/colors/blue.css" media="screen">

Setting the Top-Bar color optional

  1. Open all html files.
  2. Go to Line 81 and add dark-bar or color-bar class to the container.
<div class="top-bar">

Setting the Boxed Layout optional

  1. Open all html files.
  2. Go to Line 74 and add boxed-page class to the container.
<div id="container" class="boxed-page">

Getting the Template Online

  1. It's important to understand that you must either have your own server space or use someone elses. The server space can be purchased at many different places around the internet. You need somewhere to put the files that you purchased so others can find them online.
  2. The template folder will have several files with the extension labeled .html. This is where all your content will. It's very important to keep the css and js-plugin folders with the HTML or the HTML will not look how the design is meant to look. Along with the HTML files, you will also have four additional folders to upload: css, js, fonts, and images.
  3. Start your FTP client (I persoanlly use Filezila which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.

CSS Structure

I'm using a set of three main CSS files (style.css, responsive.css and color css file) in this theme which you are free to modify depending on the customization you require. There are also four additional style sheet that we do not recommend changing. Those style sheets reside in the 'CSS' folder. The style sheets are:

  1. style.css - The main stylesheet.
  2. responsive.css - Responsive Styles.
  3. bootstrap.css - Bootstrap.
  4. settings.css - Slider Revolution settings.
  5. fontello.css - Font Icons.

Finished color scheme template in the /css/colors/ directory:

If you would like to edit a specific section of the site, simply find the appropriate label in the style.css file, and then scroll down until you find the appropriate style that needs to be edited.

Setup Google Map

Open contact.html file and go to line 240 (contact.html)

<!-- Map -->
<div id="map" data-position-latitude="30.044300" data-position-longitude="31.235700" data-marker-img="images/marker.png"></div>
  • data-position-latitude - Set your position latitude value.
  • data-position-longitude - Set your postion longitude value.
  • data-marker-img - Set your position marker image.
Notice! To convert an address into latitude and longitude please use this converter.

Setup Contact Form

  1. Open the mail.php file.
  2. Go to Line 15-16, and replace with your E-mail.
// put your email address here
mail("zoomarts.com@gmail.com", $subject, $body, $headers);
